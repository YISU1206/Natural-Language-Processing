Final Project: Keyphrase detection and Text summarization using TextRank algorithm

Aim of the project:
Implement TextRank Algorithm to extract keyphrases and summarize text from online news corpora

Packages in my project:
1) re: package contains regular expression functions
2) networkx: return PageRank algorithm
3) itertools import combinations: return combinations
4) sys: help to execute the file using command line
5) nltk
6) nltk import tokenize, pos_tag: split text to sentences; add POS tags to words in sentences
7) nltk.corpus import stopwords: import stopwords to remove stop words like 'a' and 'the' in the sentence
8) math import log: return the natural logarithm of x (to base e)

Instructions to execute this project:
1) This program can be executed within Python 3.6 
2) Open command line and enter: 
Python NLP_Final.py <path_Of_Test_Document/docName> N M

    e.g. Python NLP_Final.py C:/Users/Helen/Desktop/test2.txt 8 8
